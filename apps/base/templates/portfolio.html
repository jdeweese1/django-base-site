A couple of my coding projects I'm very proud of are this website, a soduku solver, and my software architecture & design final project.

This site

This site is built with Django, a Python framework for web application development. The skeleton of this project comes from Brent O'Connor's Django Base Site. By making some modifications, like adding a contact form, blog, and corresponding admin forms I've made my own custom site. For deployment, for the moment I'm using Brent's recipe for Heroku.

Soduku solver

I helped build a soduku solver in C as our final project in my C lab. My part in the lab was teach my partners how to use Git in a collaborative environment. Additionally, I wrote the test framework of the core of the project, as well as helper methods. To streamline the presentation process in class, I wrote an autorunner shell script that would demo the features of our program with a single command. Our final product could solve, generate, and validate puzzles using file IO.
